# ✈️ Airline Management System

A full-stack web application built using **Angular**, **Spring Boot**, and **MySQL** to manage flight records. This system allows users to **add**, **update**, **delete**, and **view flights** through a responsive and user-friendly interface.

---

## 🔧 Tech Stack

| Layer         | Technology            |
|---------------|------------------------|
| Frontend      | Angular 16+ (Standalone Components), Bootstrap |
| Backend       | Spring Boot (Java)     |
| Database      | MySQL                  |
| Build Tools   | Maven, Node.js, npm    |

---

## 🚀 Features

- ➕ Add new flights
- 📝 Edit existing flights
- ❌ Delete flights
- 📋 View all flights in a table
- 🔄 Real-time form updates for edit operations
- ✅ Integrated Angular + Spring Boot with REST API
- 🎨 Styled using Bootstrap for a clean UI

## ⚙️ How to Run

### 1️⃣ Backend (Spring Boot)

1. Open in Eclipse or IntelliJ  
2. Create MySQL database (e.g., `airline_db`)
3. Update DB config in `application.properties`  
4. Run the app → API runs on `http://localhost:8081`

### 2️⃣ Frontend (Angular)

```bash
cd frontend/
npm install
ng serve
App runs at: http://localhost:4200
